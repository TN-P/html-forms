#  HTML Contact Form

A responsive and user-friendly **HTML Contact Form** built using **HTML and CSS**.  
This project demonstrates form design, input validation, and basic UI styling for collecting user information.

---

##  Features

- Clean and responsive form layout  
- Input fields for:
  - Name  
  - Email  
  - Phone number  
  - Message  
- Required field validation using HTML5  
- Proper input types (email, number, text)  
- Submit button with hover effects  
- Mobile-friendly design  

---

##  Technologies Used

- HTML5  
- CSS3
